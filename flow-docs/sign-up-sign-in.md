# <a name="sign-up-and-sign-in-for-microsoft-flow"></a>Iscriversi e accedere per Microsoft Flow
<iframe width="560" height="315" src="https://www.youtube.com/embed/cRkmSZrctLc?list=PL8nfc9haGeb55I9wL9QnWyHp3ctU2_ThF" frameborder="0" allowfullscreen></iframe>

Iniziare a utilizzare Microsoft Flow come utente singolo è facile. Per poter creare un flusso, iscriversi usando un indirizzo di posta elettronica. Se non si è mai usato un prodotto Microsoft online con tale indirizzo, sarà necessario impiegare qualche secondo per la registrazione.

## <a name="sign-up-free"></a>Iscriversi gratuitamente
Se non sono mai stati usati altri prodotti Microsoft online, sarà necessario iscriversi.

1. In [flow.microsoft.com](https://flow.microsoft.com), scegliere o toccare **Iscrizione gratuita** in alto a destra.
2. Immettere l'indirizzo di posta elettronica.
3. Fare clic o toccare la freccia destra.

    ![Collegamento di iscrizione](./media/sign-up-sign-in/signup.png)

## <a name="sign-in"></a>Accedi
Se sono già stati usati altri prodotti Microsoft online, per lavoro o per scopi personali, sarà sufficiente effettuare l'accesso.

1. In [flow.microsoft.com](https://flow.microsoft.com), scegliere o toccare **Accedi** in alto a destra.

    ![Collegamento di accesso](./media/sign-up-sign-in/signin.png)
2. Immettere l'indirizzo di posta elettronica.
3. Nella pagina di accesso, immettere l'indirizzo di posta elettronica e la password.

## <a name="using-paid-features"></a>Utilizzare le funzionalità a pagamento
Chiunque può iscriversi e accedere al piano gratuito di Microsoft Flow. Se la propria organizzazione ha acquistato Office 365 o Dynamics 365, potrebbero essere accessibili le funzionalità aggiuntive di Microsoft Flow. È anche possibile ottenere una versione di prova gratuita di 90 giorni oppure acquistare il piano 1 o 2 di Microsoft Flow, se si vogliono usare le funzionalità a pagamento. [Altre informazioni sulla fatturazione](billing-questions.md).

Per informazioni di tipo amministrativo, consultare le [domande frequenti su Flows nell'organizzazione](organization-q-and-a.md).
